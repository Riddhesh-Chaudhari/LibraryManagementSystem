# Library Management System (Console Based)

Console-based Library Management System built using Java and Eclipse.

## Features
- File-based login system
- Add new books
- Search books
- Issue books
- Return books
- Delete books
- Prevent duplicate book IDs

## Technologies
- Java
- Eclipse IDE
- File Handling
- OOP Concepts

## How to Run
1. Open project in Eclipse
2. Run Main.java
3. Use users from users.txt
